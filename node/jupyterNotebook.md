# Jupyter Notebook for JavaScript

- 2018/08/21
- [ijavascript](https://github.com/n-riesco/ijavascript)


## 環境

- Windows 10 專業版
- Python 3.6.0 :: Anaconda 4.3.0 (64-bit)

```sh
### 建立虛擬環境
$ python -m venv jbook

# 安裝阿~~~
(jbook)$ conda install nodejs
(jbook)$ npm install -g ijavascript
(jbook)$ ijsinstall

(jbook)$ jupyter notebook
# 安心使用~
```
