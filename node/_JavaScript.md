
# JavaScript

  - number
  - string
  - boolean
  - null
  - undefined
  - object
  - array
  - symbol (於 ES6 加入)
- 自動插入分號(ASI, automatic semicolon insertion)


# Function

```js
// 函式定義(FD, function declaration)
function sum(a, b) {
  return a + b;
}


// 函式表達式(FE, function expression)
const sum = function(a, b) {
  return a + b;
}


// 箭頭函式(arrow function)
const sum = (a, b) => a + b;


// 
```
